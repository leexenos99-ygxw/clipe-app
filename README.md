📖 项目介绍
Clipe 是一款面向学生、教师的轻量化AI教育效率工具，主打试卷智能拆分能力。依托AI图像理解与文档分析技术，可自动解析整张试卷图片，精准识别题号、题目区域、图文边界、选项归属，一键拆分独立试题，彻底替代手动截图、裁剪、整理的重复操作，高效助力错题整理、题库搭建、教学素材制作。
区别于传统仅提取文字的OCR工具，Clipe 核心突破是从“识别文字”升级为“理解试卷结构”，完美适配中小学、初高中各类试卷、练习册、真题资料的拆分场景。

🎯 项目背景
在日常学习与教学场景中，整理试卷错题、优质考题是高频需求，但传统整理方式效率极低：
拍照 → 手动定位题目 → 截图裁剪 → 保存分类 → 整理归档
一张包含20-30道题的试卷，人工整理往往需要10-20分钟，大量时间消耗在重复机械操作上，严重挤占深度学习、刷题复盘的时间。
Clipe 诞生于真实学习痛点，核心宗旨：把整理试卷的时间，还给真正的学习与思考，用AI自动化替代重复劳动，提升学习与备课效率。

---
🚀 核心功能
1. AI 智能一键拆题
上传整张试卷图片，系统自动完成图像优化、版面分析、题目区域检测，精准识别每一道题的独立范围，批量拆分所有试题，无需逐题手动操作，适配黑白试卷、彩色试卷、真题、练习册等各类素材。
2. 可视化智能编辑微调
AI自动生成题目边界框，支持鼠标拖拽调整选区、自定义裁剪范围、实时预览拆分效果。针对复杂排版（跨行题目、图文混合、表格题型），可快速人工修正误差，保证拆分精准度。
3. 多格式试题导出
支持单题PNG高清图片导出、整页试题PDF打包导出，可直接用于：学生错题本整理、个人专属题库搭建、教师备课素材制作、复习资料汇编。

🧠 技术架构
项目采用轻量化AI文档理解流水线，全流程结构化处理，兼顾效率与精度：
上传试卷图片 → 图像质量增强处理（降噪、提亮、矫正角度） → 文档版面结构分析 → 题目区域智能检测 → OCR文字识别 → 题目结构化重建（关联图文、选项、题干） → 单题独立导出

💻 本地部署
环境要求
- Node.js &gt;= 18.0
- npm >= 9.0 或 yarn 最新版
部署步骤
1. 克隆项目仓库
git clone https://github.com/yourname/clipe.git
2. 进入项目目录
cd clipe-frontend
3. 安装项目依赖
npm install
4. 本地启动开发环境
npm run dev
5. 项目打包部署
npm run build
本地访问地址
启动成功后，默认访问：http://localhost:5173

❓ 常见问题 FAQ
Q1：支持哪些图片格式？
支持 JPG、PNG、JPEG 主流图片格式，建议上传清晰、无反光、无倾斜的试卷图片，识别效果更佳。
Q2：AI拆分存在误差怎么办？
工具支持手动拖拽调整题目边界框，可自由裁剪、修正AI识别误差，适配所有特殊排版题型。
Q3：所有功能是否完全免费？
当前所有拆题、编辑、导出功能完全免费，无次数限制、无水印、无需登录。
Q4：是否支持手机端使用？
项目适配移动端响应式布局，手机、平板、电脑均可流畅访问使用。
